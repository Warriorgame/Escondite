# Cómo arrancar HideGround v4

```bash
npm install
npm run dev
```

Abre la URL que salga en terminal. Normalmente será:

```txt
http://localhost:8080
```

DevLab:

```txt
http://localhost:8080/dev-tools
```

Si Supabase sigue devolviendo 400/409, ejecuta en Supabase > SQL Editor:

```txt
supabase/supabase_hideground_repair_v4.sql
```
